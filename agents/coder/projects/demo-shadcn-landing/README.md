# Demo Shadcn Landing Page

Demo de mini-site local en Vite/React + UI Shadcn, effets "wow" (canvas, framer-motion, morphing SVG/CSS).
- `npm run dev` pour lancer localement
- Structure modulaire, zéro backend/persistance

Contient :
  - HeroSection (bg dynamique)
  - FeatureGrid (cartes animées)
  - InteractiveShowcase (mini-démo)
  - CallToAction (CTA animé)
  - AnimatedBackground, Navbar, Footer
  - Thème Shadcn (mode dark/light, accents dynamiques)
  - Animations Framer Motion & micro-interactions
